To install:
- copy server_config.json and topology.json to /etc/HectorTa1989/ and edit them with your configuration
- copy white_list.txt and black_list.txt to /etc/HectorTa1989/ (they can remain empty)
- copy oui.txt to /etc/HectorTa1989/

In server_config.json, blind_mode and using_sensor are important options. See main README to understand how to use them.
